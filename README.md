# Flop Mine

## 介绍
Flop Mine是 [扫雷网](http://www.saolei.wang/Main/Index.asp) 的 Android 客户端（非官方），录像及用户数据均来自 [扫雷网](http://www.saolei.wang/Main/Index.asp)。

### 主要功能

* 扫雷网数据爬取、排版
* 扫雷网录像播放
* 选择本地录像进行播放
* 开始新游戏
* 排序筛选（不同页面可根据神界、人界、初级、高级、Bv等作为筛选排序依据）
* 自定义部分页面的默认用户ID
